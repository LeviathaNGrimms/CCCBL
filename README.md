# LeviathaN's Custom Community Center Bundles Loader / CCCBL

A unified framework for custom Community Center bundle packs in Stardew Valley.

**Download on [Nexus Mods](#)** (link here once published)

## Building from source
1. Install the .NET 6 SDK
2. Create a `Directory.Build.props` file with your game path:
```xml
<Project>
  <PropertyGroup>
    <GamePath>PATH\TO\Stardew Valley</GamePath>
  </PropertyGroup>
</Project>
```
3. Run `dotnet build`
